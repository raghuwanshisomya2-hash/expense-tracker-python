# Expense Tracker (Python)

This project is a simple command-line Expense Tracker built using Python. It helps users record daily expenses and analyze spending patterns.

## Features
- Add daily expenses
- View all recorded expenses
- Calculate total expenses
- Category-wise expense summary
- Data stored using file handling

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Run `expense_tracker.py`
3. Follow on-screen instructions
